```

./build/index_fund_monitor --help
./build/index_fund_monitor --version
./build/index_fund_monitor --unknown
```

**验证结果**：
- 编译成功，零警告
- 无参数 → 显示帮助，退出码0
- --help → 显示帮助，退出码0
- --version → 显示版本号 0.1.0，退出码0
- --unknown → 标准错误输出错误信息，退出码1
