# A股指数ETF监控器：Agent 开发任务提示词

## 项目范围

第一版只支持：

- 上海证券交易所和深圳证券交易所上市的股票型指数ETF
- ETF基础信息
- ETF与跟踪指数映射
- 指数PE TTM
- 指数PB
- 指数股息率
- 历史估值百分位
- 条件筛选
- 本地终端预警
- CSV导出

第一版不支持：

- 港股
- 美股
- 场外指数基金
- 债券ETF
- 商品ETF
- 跨境ETF
- 杠杆ETF
- 反向ETF
- 实时行情
- 自动交易
- 图形界面
- Web服务
- 投资建议
